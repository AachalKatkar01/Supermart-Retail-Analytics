# Supermart Retail Analytics

## 📌 Project Overview
This project analyzes retail sales data to generate business insights for CEO and CMO decision-making.

## 📊 Key Insights
- Time Series Analysis of Revenue
- Top Countries by Revenue
- Top Customers by Revenue
- Regional Demand Analysis (Excluding UK)

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Data Visualization Techniques

---

This project demonstrates data analysis and business intelligence skills using retail sales data.
